# City - Smart

Plataforma de una ciudad inteligente desarrollada por BeeStudents utilizado Tegnologías para FrontEnd, UI/UX y acceco a algunas API´s públicas 

## Caracteristicas 

- Login y Registro de Usuarios. 
- Dashboard Responsivo. 
- Integracion de Firebase (Pasos Evolutivos).
- Consumo de API´s Públicas (Clima, Noticias, Calidad de Aire).

## Tecnologías Utilizadas (Stack de Desarrollo)

- HTML5.
- CSS3.
- Bootstrap 5.
- JavaScript.
- Firebase.
